基本查询

1. 基本查询

1.1 什么是SQL

​ 结构化查询语句，是一种数据查询和程序设计的语言。用于存取数据、更新和管理关系数据库系统

1.2 SQL的分类

  • DCL(Data Controll Language 数据控制语言): grant 授权, revoke 解除权限
  • DDL(Data Definition Language 数据定义语言): create, alter, drop, truncate
  • DML(Data Manipulation Language 数据操作语言): insert, update, delete
  • DQL(Data Query Language 数据查询语言) select

1.3.4 null空值问题

什么是空值?

​ 未指定的，不可预知的，无效的，既不是0，也不是空字符串

  • 空值参与的表达式，返回结果都为空
  • null!=null 空值永远不为空

3. 排序

order by 列名 asc（升序）|desc（降序）

示例：

 
xxxxxxxxxx
6
1

2

3
--按入职时间升序，工资降序
